# AI Agent Practice

## What I built
- **Tool Calling** - LLM se weather, time, calculator tools use karna
- **RAG** - ChromaDB se documents retrieve karke LLM ko context dena  
- **LangGraph** - Nodes, edges aur conditional routing se agent flow banana
- **Conditional Edges** - If/else type routing jo decide kare kaunsa node aage jayega
- **Short Term Memory** - Conversation history maintain karna
- **Long Term Memory** - ChromaDB persistent storage se sessions ke beech memory
- **FastAPI** - Agent ko web API mein wrap kiya — browser se access karo
- **Docker** - Agent ko containerize kiya — kisi bhi machine pe chalega
  

## Tech Stack
- Python, Groq API, LangGraph, ChromaDB, Sentence Transformers, FastAPI, Uvicorn
